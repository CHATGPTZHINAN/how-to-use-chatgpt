    # 如何注册 chatgpt？都 2024 年了，你还没有用上 chatgpt？chatgpt 注册保姆级教程

    # 前言

    由于现在注册 chatgpt不需要手机验证码，所以方便多了，之前还需要去接码平台进行手机验证码验证，现在是通过邮箱就可以进行注册了，方便多了。

    # 提前准备

    • 科学上网工具
    • 谷歌邮箱

    # 1.科学上网工具

    如果你连科学上网都不会，那 chatgpt 肯定也用不上了

    # 2.谷歌邮箱

    如果你没有谷歌邮箱，可以注册一个，毕竟使用很多墙外的服务都可以通过谷歌邮箱进行第三方进行验证，注册之后挺方便的。

    首先点击这个链接谷歌邮箱 (opens new window)

    点击下图中的创建账号

    接下来输入你的姓名，最好是英文，如果你不想用你的真实的名字的话，可以随便取一个名字

    接下来输入出生年月、性别，也可以随便输入，不过最好设置的年龄为年满 18 岁

    接下来可以设置自己的谷歌邮箱地址，不能和已有的重复

    接下来就是设置密码

    最后就是使用自己的电话接收一个验证码就行了，注意需要选择中国

    至此，你就成功注册了一个谷歌邮箱账号

    # 3.注册 chatgpt

    打开这个链接chatgpt 注册 (opens new window)

    点击 sign up，进入下面这个页面，输入刚刚注册的谷歌账号，如果你使用的谷歌浏览器，可以直接点击下方的继续使用 Google,我就是这样做的

    当然也可以输入邮箱账号和密码

    至此 就成功注册了 chatgpt，可以开心的玩耍了

    有可能会出现下面的状况 你可以切换自己的梯子节点为美国 多试几次就好了

    如果你想升级 chatgpt3.5 到 4，可以参考这篇文章，毕竟 4 比 3.5 更加强大

    chatgpt4.0 如何升级？chatgpt 升级到 4.0 保姆级教程，五分钟包教包会 (opens new window)
