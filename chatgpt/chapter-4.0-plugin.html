    # 分享十个好用的 chatgpt 4.0 插件

    最近将 chatgpt 升级到了 4.0，发现它的功能更加强大了，这里你可以看一下我之前做的测评，将 chatgpt4.0 与其它大语言模型进行智力对比。现在 chatgpt4 可以安装插件了，可以说就像给了他一堆武器库，随意挑选，差什么能力，通过插件就可以补齐了。

    那么，插件的官方定义是什么，这里我问了一下 chatgpt4，我们来看看管方解答，hhh

    可以看到，插件可以让 chatgpt 实时联网，所以它的训练数据就不仅仅局限于被拿来训练时候的数据，而是可以实时从网上获取最新的数据。可以调用第三方数据库和工具箱，扩展自己的能力，这样看插件是不是有点像哆啦 A 梦的百宝箱。接下来就分享我觉得有用的几个插件，如果你还不知道怎么升级到 chatgpt4，可以参考一下这篇文章：chatgpt4.0 如何升级？chatgpt 升级到 4.0 保姆级教程，五分钟包教包会 (opens new window)

    # 如何使用

    如图，升级 chatgpt4 之后，点击 explore gets，然后在输入框中搜索自己需要的插件就可以。

    # 最强水论文插件 —— MixerBox Scholar

    现在到了本科生、研究生写论文的时候，有了这个插件，再也不用怕论文的字数要求了，我们先让 ai 写 500 字的背景与意义让我们参考一下。当然这只是一个简单示范，如果要写的更加像样，需要加更多的限定词。不过我们可以做为参考嘛，不至于到写的时候，半个小时憋不出一个字来。

    # 联网搜索最强 —— WebPlot

    WebPilot 插件是一种高级的 AI 工具，专为 ChatGPT 用户设计，用于增强其在线浏览和内容生成能力。话不多说，让我们直接看效果。可以看到，就是一个加强版的谷歌，还具有文生图的能力，但是你可以以对话的形式和它聊天，并对返回结果进行要求，需要注意的是，默认返回的是英文，如果需要中文，可以在询问的时候加一个限定词。

    # 计算、画图、社科 —— Wolfram

    Wolfram 插件是为 Wolfram 语言和 Mathematica 软件开发的插件。Wolfram 语言是一种高级编程语言，主要用于科学、工程、数学和计算领域。对于数学专业的学生有福音了。

    # 学习路径（个人导师） —— Tutory

    这个插件可以为个人制定学习路径，我让他为我制定一个前端学习计划，看起来确实有模有样，但是感觉和网上的学习路线比，确实不够细节详细，不过网上的都是各位大佬花了许多时间收集整理的。

    # 总结

    以上就是我目前探索的四个插件，感觉水论文插件可以为现在写论文的学生做为参考借鉴，另外的联网搜索插件也不错，还具有文生图的能力。
